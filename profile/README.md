## What is Prodigy Origin?

Prodigy Origin is a browser extension that mods the Prodigy Math Game to give players access to everything the game has to offer. No subscriptions, paywalls no just the full Prodigy experience, for free. 

**Features include:**
- Unlock all items, pets, mounts, and gear
- Access members-only areas and content
- Quality-of-life improvements and UI tweaks
- Lightweight, fast, and regularly updated
